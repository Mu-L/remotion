We offer three components for embedding videos into your Remotion composition:

- [`<OffthreadVideo />`](/docs/offthreadvideo) - **our recommendation**, based on a Rust-based frame extractor
- [`<Video />`](/docs/video) from `remotion` - based on the HTML5 `<video>` element
- [`<Video />`](/docs/media/video) from `@remotion/media` - an experimental WebCodecs-based component, set to become the default

This page offers a comparison to help you decide which tag to use.

|                                          | [`<OffthreadVideo />`](/docs/offthreadvideo)                                                     | [`<Video />`](/docs/video) <br/> [`<Audio />`](/docs/audio) <br/> (`remotion`) | [`<Video />`](/docs/media/video) <br/> [`<Audio />`](/docs/media/audio) (`@remotion/media`)                             |
